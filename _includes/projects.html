Projects

Decahose Data Management Platform
  • Designed, developed and produced Decahose Data Management Platform. Successfully implemented in Serendio.
  • This architecture ingests high volume of twitter stream (5000 tweets/min), enrich it on the fly, expose the stored data via REST API.
  • Technology stack : Python, Java, Kafka, Storm, Cassandra, Elasticsearch, Monit.
EmotoData
  • Played a major role in development of an architecture to handle streaming data from Chinese websites, implemented in Serendio.
  • This service streams data from various Chinese websites like sina weibo, tencent, renren, etc. Enriches and stores them.
  • Technology stack : Python, Kafka, Cassandra, ElasticSearch, Solr.
Smart switch (Internet of Things)
  • Developed a multi pin socket box (spike buster), where devices connected in each sockets can be toggled on/off from Internet. A responsive web app has interface to toggle and monitor the energy readings of each devices connected in socket. Implemented in a hackathon.
  • Technology stack : Python, Flask, Arduino, Microcontrollers, SQLite.
Smart pill box (Internet of Things)
  • Developed a 4 compartment pill box that is connected to Internet. Once the details of pills in each compartment is setup in the web interface, It reminds at the respective time to take particular pill, Sends an email to place order when specific pill gets over. Implemented as hobby project.
  • Technology stack : Python, Flask, Arduino, Microcontrollers, SQLite.
